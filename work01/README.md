# Dalong MS 串口控制 GUI

这是一个用于控制 Dalong 磁力搅拌加热器的 Python 工具，支持串口连接、自动握手、转速与温度设置、实时状态刷新和运行日志。

## 1. 安装依赖

```bash
pip install -e .
```

如果你不使用 `pyproject` 安装方式，也可以直接安装：

```bash
pip install pyserial
```

## 2. 启动界面

```bash
python main.py
```

## 3. 使用步骤

1. 选择串口与波特率，点击“连接”。
2. 连接后程序会自动握手，也可以点击“握手测试”手动确认。
3. 输入转速和温度，分别点击“设置搅拌”“设置加热”。
4. 需要紧急停机时点击“一键停止”。

界面会每秒刷新一次设备状态。

## 4. 本版优化点

1. 串口收发增加校验与帧过滤，减少误读和脏数据干扰。
2. 连接状态颜色提示，控制按钮会随连接状态自动启用/禁用。
3. 控制区改为可调步进输入（Spinbox），减少手输错误。
4. 增加“立即刷新”按钮和运行日志面板，便于排查握手或控制异常。
