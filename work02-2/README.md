# work03-2: PTA9B01 PT100 RS485 控制台

基于 PTA9B01 PT100 手册中的信息实现：

- 通讯协议：Modbus RTU
- 功能码：03 读、06 写
- 串口默认：9600, N, 8, 1
- 支持：读取温度、读取 PT100 原始值、写从站地址（可配置地址寄存器）

> 注意：你提供的 PDF 中未包含完整寄存器地址表（文档写明需参考独立协议文档），
> 所以界面做成“寄存器可配置”模式。默认值是常见配置：温度寄存器=0，PT100寄存器=1，地址寄存器=2。

## 运行 (uv)

在仓库根目录执行：

```bash
uv run work03-2/main.py
```

## 界面说明

1. 串口配置
- 选择 COM 口
- 设置波特率（默认 9600）
- 设置超时（默认 0.35s）
- 点击“连接”

2. 读数配置
- 从站地址：默认 1
- 温度寄存器：默认 0
- PT100值寄存器：默认 1
- 温度缩放因子：默认 10（即寄存器值 253 -> 25.3C）
- 温度按有符号16位解析：支持负温

3. 读取模式
- 点击“读取一次”获取当前值
- 勾选“轮询”按设定间隔自动读取

4. 写从站地址
- 配置“地址寄存器”和“新地址”
- 点击“写入地址”
- 成功后建议设备断电重启再连接

## 排障建议

- 若读取失败：先确认 A/B 线、GND、供电和 USB-RS485 转换器驱动。
- 若 CRC 错误：常见原因是串口参数不匹配或总线干扰。
- 若一直无响应：请确认从站地址、寄存器地址和传感器型号是否一致。
