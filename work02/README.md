# LCUS-8 两路继电器 GUI 控制

本项目基于 LCUS-8 USB 继电器模块协议，实现了一个桌面 GUI（Tkinter），用于控制第 1 路和第 2 路继电器。

## 功能

- 自动枚举并选择串口（COM）。
- 连接/断开串口。
- 控制第 1 路、第 2 路开关。
- 发送 `FF` 查询继电器状态，并显示前两路状态。
- 一键关闭 1/2 路。

## 协议（来自设备手册）

- 波特率：`9600`
- 打开第 1 路：`A0 01 01 A2`
- 关闭第 1 路：`A0 01 00 A1`
- 打开第 2 路：`A0 02 01 A3`
- 关闭第 2 路：`A0 02 00 A2`
- 查询状态：`FF`，返回 8 字节（`01`=开，`00`=关）

程序中按文档规则计算校验：`checksum = (0xA0 + channel + state) & 0xFF`。

## 运行

1. 安装依赖：

```bash
pip install pyserial
```

2. 启动：

```bash
python main.py
```

## 说明

- 当前 GUI 只操作第 1/2 路，但底层已支持 1~8 路指令格式。
- 若查询状态超时，请确认串口号、波特率、电源和 USB 连接是否正常。
